Import dummy data for themes developed by Keon Themes.

== Description ==
A easy plugin to import dummy data for themes by Keon Themes.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Keon Toolset
3. Activate Keon Toolset plugin from your Plugins page.
4. Go to the Appearance -> Demo Import

= From WordPress.org =

1. Download keon-toolset.
2. Upload the 'keon-toolset' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc.)
3. Activate Keon Toolset from your Plugins page.

== Frequently Asked Questions ==
1. Where I can see the Settings?
Check all the available options inside Appearance > Demo Import
